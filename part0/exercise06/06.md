    Browser->>Browser: Creates a note object with content and date

    activate Server
    Server-->>Browser: 201 Created response (note saved)
    deactivate Server

    Browser->>Server: GET /data.json (fetches updated notes)
    activate Server
    Server-->>Browser: Sends the list of notes in JSON (including the new one)
    deactivate Server

    Note right of Browser: JavaScript updates the UI without reloading the page
